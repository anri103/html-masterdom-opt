| Группа     | Пример                    | Назначение                            |
| ---------- | ------------------------- | ------------------------------------- |
| `layout`   | `header`, `footer`        | постоянные на всех страницах          |
| `sections` | `hero`, `benefits`, `faq` | большие секции на отдельных страницах |
| `ui`       | `product-item`, `button`  | мелкие переиспользуемые элементы      |
